# Zoe — Hiragana & Katakana Quiz

A clean, minimalist quiz app for learning Japanese kana.

## Features

- **3 modes**: Hiragana only, Katakana only, or Mixed
- **2 question types**: Kana → Romaji or Romaji → Kana
- **Adjustable length**: 10, 20, or all 46 characters
- **Instant feedback** with correct/wrong highlighting
- **Score summary** at the end

## How to use

Just open `index.html` in any browser — no installation needed.
